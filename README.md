# Óticas Fácil com Supabase

## 1. Instalar
npm install

## 2. Criar arquivo .env
Crie um arquivo `.env` na raiz usando o modelo do `.env.example`.

## 3. Rodar localmente
npm run dev

## 4. Publicar no Vercel
Adicione as variáveis:
- VITE_SUPABASE_URL
- VITE_SUPABASE_ANON_KEY

## 5. Banco
Cole o conteúdo de `supabase-schema.sql` no SQL Editor do Supabase.
