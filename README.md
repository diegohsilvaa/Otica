# Óticas Fácil - Fechamento de Caixa

Nova versão com:
- visual mais empresarial
- fechamento diário de caixa
- bloqueio de edição/apagamento após fechamento

- resumo gerencial por responsável

## Importante
Rode o arquivo `supabase-schema.sql` no Supabase para criar a tabela:
- lancamentos_caixa
- fechamentos_caixa

Depois publique novamente no projeto.
